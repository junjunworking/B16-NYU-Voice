Django Application launch Guide

Follow these instructions to install and launch this Django application from a .zip file.

1. Make sure Django and Python are installed before running this project.

2. Extract the .zip file containing your Django application to your desired directory.

3. Open a new terminal, run the following command:
   $ python manage.py runserver

4. Access the application by the address provided in the terminal.

5. To stop the development server, press Ctrl + C in the terminal.
